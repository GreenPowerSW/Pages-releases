GroupCellBalancing

Cell Balancer Group Control & Monitoring System

About

GroupCellBalancing is a WPF application for controlling and monitoring Cell Balancer equipment. It manages up to 40 channels (5 Boards x 8 Channels) with group-based independent operation, real-time voltage/current monitoring, and data logging.
